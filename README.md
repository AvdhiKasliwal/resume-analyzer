# 🚀 AI Resume Analyzer

## 📌 Overview
An AI-powered web application that analyzes resumes and compares them with job descriptions.

## 🔧 Features
- Resume upload (PDF)
- Skill extraction
- Missing skill detection
- Match score using AI (TF-IDF + Cosine Similarity)
- Smart suggestions

## 🛠️ Tech Stack
- Python
- Flask
- HTML/CSS
- Scikit-learn
